# Código mobile

## Variáveis de ambiente

Crie um arquivo `.env` na raiz de `mobile/` usando o modelo em `.env.example`.

- `GOOGLE_WEB_CLIENT_ID`: client ID do tipo Web (o mesmo configurado no backend em `GOOGLE_CLIENT_ID`).
- `API_URL`: URL base da API usada pelo app mobile. Se não definir, o padrão é `http://localhost:8080`.
