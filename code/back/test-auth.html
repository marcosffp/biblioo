Não autenticado

Criar conta local

Cria uma conta com e-mail e senha. Você ficará logado ao final.

Login com e-mail e senha

Retorna accessToken + refreshToken. Salvo no localStorage.

Login com Google

Cole um Google ID Token obtido via OAuth Playground ou via google.accounts.id.renderButton no seu frontend.

Esqueci minha senha

Envia um e-mail com link de redefinição. Conta Google retorna mensagem específica. E-mail inexistente retorna resposta genérica (sem revelar existência).

Redefinir senha

Token preenchido automaticamente se vier pelo link do e-mail (?token=xxx na URL). Link é de uso único e expira em 30 min. Encerra todas as sessões ativas ao concluir.

Criar senha — conta Google

Exclusivo para contas vinculadas ao Google que ainda não têm senha. Requer autenticação — faça login com Google primeiro.
