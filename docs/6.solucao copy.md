# 6. Projeto da Solução

Esta seção apresenta as interfaces desenvolvidas para o sistema Biblioo, com descrição sucinta de cada tela e os requisitos funcionais que cada interface atende, conforme definidos na seção [3. Requisitos](3.requisitos.md) e nas histórias de usuário da seção [4. Modelagem](4.modelagem.md). Requisitos implementados em ambas as plataformas são apresentados em seção única, com capturas das interfaces web e mobile. Funcionalidades exclusivas de uma plataforma são indicadas como _(exclusivo web)_ ou _(exclusivo mobile)_.

---

### 6.1 Busca de Livros

![](https://github.com/user-attachments/assets/893999e6-c38e-43d2-9fad-3d09e58e99b3)

*Figura 1 - Busca de Livros (Web)*

<table><tr>
<td><img width="301" height="655" alt="" src="https://github.com/user-attachments/assets/9f041bd4-56d6-46bc-8970-1fc7dc58fd83" /><br><em>Figura 2 - Busca de Livros (Mobile)</em></td>

</tr></table>

Tela de busca do catálogo, acessível pela barra de pesquisa global. Ao pesquisar por título, autor ou ISBN, o sistema retorna resultados do acervo interno e, quando necessário, consulta fontes externas. Na web, um modal exibe capa, título, autores, número de páginas, sinopse e botão "Adicionar na estante". No mobile, a busca leva à tela de detalhe do livro com as mesmas informações e ações.

**Requisitos atendidos:** RF-01

**História de usuário:** HU-01

### 6.4 Solicitação para Seguir Perfil Privado

![](https://github.com/user-attachments/assets/217807b1-bf04-4b3f-a024-9c1805065936)

*Figura 8 - Solicitação para Seguir Perfil Privado (Web)*

<img width="301" height="655" alt="" src="https://github.com/user-attachments/assets/4fddc8d1-f453-4c9d-b79e-80715fcca628" />

*Figura 9 - Solicitação para Seguir Perfil Privado (Mobile)*

Ao tentar seguir um leitor com perfil privado, o visitante vê apenas nome, username e contadores de seguidores, enquanto o restante do conteúdo permanece oculto. O botão "Solicitar para seguir" (web) / "Pedir para seguir" (mobile) envia o pedido, que fica pendente até ser aceito ou recusado pelo dono do perfil.

**Requisitos atendidos:** RF-04

**História de usuário:** HU-04
