# 6. Projeto da Solução

_Apresente as telas dos sistema construído com uma descrição sucinta de cada uma das interfaces._

Esta seção apresenta as interfaces desenvolvidas para o sistema Biblioo, com descrição sucinta de cada tela e os requisitos funcionais que cada interface atende, conforme definidos na seção [3. Requisitos](3.requisitos.md) e nas histórias de usuário da seção [4. Modelagem](4.modelagem.md).

---

## 6.1 Tela de Login

![](https://github.com/user-attachments/assets/b252a247-c6b3-4771-b40d-f8e1dd5197aa)

*Figura 1 - Tela de Login*

Tela inicial do sistema, com layout dividido em duas colunas: a esquerda exibe o slogan da plataforma com identidade visual em verde, e a direita contém o formulário de autenticação com campos de e-mail e senha, botão de acesso e opção de login via Google.

**Requisitos atendidos:** RF-03 · RF-34

---

## 6.2 Tela de Cadastro

![](https://github.com/user-attachments/assets/d4e21f26-c9da-4dfd-a445-04be5268979b)

*Figura 2 - Tela de Cadastro*

Formulário de criação de nova conta, com campos para nome de usuário, e-mail, senha e confirmação de senha. Também permite o cadastro rápido por meio da conta Google. Ao final, oferece redirecionamento para a tela de login para usuários já cadastrados.

**Requisitos atendidos:** RF-02 · RF-34

---

## 6.3 Onboarding de Preferências

![](https://github.com/user-attachments/assets/d8a606c4-0281-4e1b-9ed1-9a3872c8cf4f)

*Figura 3 - Onboarding: Escolha de Gêneros*

![](https://github.com/user-attachments/assets/82fe1c4d-6ab1-4bd7-8540-f7912eb47edd)

*Figura 4 - Onboarding: Livros Favoritos*

Fluxo de personalização exibido no primeiro acesso, dividido em dois passos. No primeiro, o usuário escolhe ao menos três gêneros de interesse entre dezesseis opções apresentadas em grade com ícone e nome — um contador na parte inferior indica quantos foram selecionados. No segundo passo, o usuário pode buscar livros que já amou por título, autor ou ISBN para turbinar as recomendações; esta etapa é opcional e pode ser pulada. Ao concluir, o botão "Começar a explorar" leva à tela principal.

**Requisitos atendidos:** RF-16

---

## 6.4 Busca de Livros
<img width="1465" src="https://github.com/user-attachments/assets/893999e6-c38e-43d2-9fad-3d09e58e99b3" />

*Figura 5 - Busca de Livros*

Tela de busca do catálogo, acessível pela barra de pesquisa global. O leitor pesquisa por título, autor ou ISBN e os resultados são exibidos em lista com capa, título, autor e nota média. A busca utiliza o índice OpenSearch para alta performance e consulta a Google Books API como fonte de fallback quando o livro não está no acervo interno, persistindo automaticamente os novos registros para buscas subsequentes.

**Requisitos atendidos:** RF-01
