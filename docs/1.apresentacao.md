# 1. Apresentação

O hábito da leitura é fundamental, mas atuamente, leitores ativos brasileiros enfrentam desafios para mantê-lo devido à falta de conexão e de uma experiência digital significativa. Métodos convencionais de acompanhamento literário não são suficientes para sustentar a motivação.

Plataformas de organização de leitura ajudam a catalogar livros e compartilhar avaliações. Entretanto, ainda apresentam limitações, como recomendações isoladas pouco alinhadas ao perfil individual e uma experiência de interação superficial.

Nesse contexto, o Biblioo propõe uma plataforma digital de comunidade literária web e mobile que transforma o histórico de leitura em uma experiência inteligente, social e contínua. Além das funcionalidades tradicionais de registro e estante virtual, a ferramenta também integra recomendações personalizadas com explicações transparentes, uma comunidade ativa em tempo real com discussões dinâmicas e a visualização clara do perfil literário do usuário (DNA Literário), proporcionando uma experiência imersiva e integrada para o leitor.


## 1.1. Problema

Os métodos atuais de registro e interação literária apresentam limitações em como o hábito de leitura é encorajado e mantido a longo prazo. Os principais desafios incluem:

- A perda de motivação do leitor ativo para continuar lendo após finalizar um livro.
- Recomendações genéricas em aplicativos atuais, não refletindo o perfil individual.
- Plataformas que entregam baixa interação nas discussões, não gerando um senso real de comunidade.
- Fragmentação da comunicação, forçando conversas literárias a migrarem para outras redes sociais e aplicativos externos.

## 1.2. Objetivos do trabalho

**Objetivo Geral:**  
Desenvolver uma plataforma de comunidade literária web e mobile para leitores ativos brasileiros, transformando o histórico de leitura em direcionamento claro, conexões reais entre leitores e participação em uma comunidade ativa.

**Objetivos Específicos:**  

- Implementar recursos de organização de estantes de forma personalizada com histórico de leitura (datas, progresso e conclusão).
- Desenvolver um sistema de recomendações personalizadas que explicam o motivo de cada sugestão baseada no perfil e comportamento do leitor.
- Construir um ambiente de interação social contínua com feed (curtidas e comentários), chats em tempo real e clubes do livro.
- Elaborar e exibir a identidade literária do usuário (o "DNA Literário"), demonstrando os padrões de leitura e preferências em um perfil visual integrado.

## 1.3. Definições e Abreviaturas

| **Termo / Sigla** | **Definição**                                                                                                                                      |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| DNA Literário     | Perfil literário visual do usuário gerado pela plataforma, combinando gêneros, autores recorrentes e padrões de leitura para expor sua identidade. |
| BookTok           | Comunidade literária ativa dentro da plataforma de vídeos curtos TikTok.                                                                           |
| API               | Application Programming Interface. Interface utilizada para comunicação entre sistemas, como o Google Books API para dados dos livros.             |
| MVP               | Minimum Viable Product (Produto Mínimo Viável). Versão inicial da plataforma com foca em funcionalidades básicas e validação de valor.             |
| Freemium          | Modelo de negócios no qual funcionalidades essenciais são gratuitas e recursos avançados ou de destaque são disponibilizados mediante assinatura.  |
