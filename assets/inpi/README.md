Este diretório contém os documentos institucionais relacionados à propriedade intelectual do projeto Biblioo, exigidos pelo Núcleo de Inovação Tecnológica (NIT) da PUC Minas junto ao INPI (Instituto Nacional da Propriedade Industrial).

## Contexto

Os projetos desenvolvidos no âmbito da PUC Minas podem gerar ativos de propriedade intelectual. O NIT atua como intermediário entre a universidade e o INPI, orientando e formalizando os acordos sobre titularidade, sigilo e eventuais registros de software. Os documentos neste diretório cumprem essa exigência institucional para o projeto Biblioo.
